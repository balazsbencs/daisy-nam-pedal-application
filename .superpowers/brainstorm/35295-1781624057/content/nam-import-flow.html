Flow summary

  1. User clicks "Import .nam / .namb" — file picker opens, accepts both extensions
  2. If .namb selected → import directly (existing behavior)
  3. If .nam selected → slim factor dialog appears (pre-set to 1.0)
  4. User adjusts slider if desired, clicks "Convert & Import"
  5. Rust runs nam2namb --slim <factor> input.nam sidecar, places result in library
  6. Toast: "Imported 'my-model' (converted from .nam)"
